# source for bsou.io


- [Elm client](https://github.com/bsouthga/blog/tree/master/src/frontend)
- [TypeScript server](https://github.com/bsouthga/blog/tree/master/src/backend)

## Requirements

- NodeJS: ^6.0.0
- awsebcli: ^3.9.0
- yarn: ^0.18.1
- elm: ^0.18.0

## Build and Deploy

After installing the above Requirements...

```shell
elm package install
yarn
npm run deploy
```
